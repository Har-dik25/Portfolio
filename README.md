# Hardik Choudhary — Full-Stack Developer

Portfolio website built with React 19, Vite 7, and styled with Vanilla CSS.

**[Live Demo](https://hardikchoudhary.vercel.app)**

## Features
- 🚀 Modern, responsive design
- 💫 Animations powered by `anime.js`
- 🎯 SEO optimized with meta tags and structured data
- 🌓 Dark/Light mode support (Coming soon)
- 📬 Contact form integration (Coming soon)

## Tech Stack
- **Frontend:** React 19, Vite 7
- **Styling:** Vanilla CSS
- **Animations:** anime.js
- **Deployment:** Vercel

## Getting Started

### Prerequisites
- Node.js (Latest LTS recommended)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Har-dik25/Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
